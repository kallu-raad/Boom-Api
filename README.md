How It Works

The API sends OTP requests to multiple Bangladeshi services:

1. RabbitHoleBD
2. Osudpotro
3. SwapBD
4. RangsEcom
5. BlackTeam
6. DeepToPlay
7. Paperfly
8. GrameenphoneBKShop
9. EasyComBd
10. RedX
11. Bikroy
12. Cinematic

Each service receives multiple requests as defined in the configuration.
